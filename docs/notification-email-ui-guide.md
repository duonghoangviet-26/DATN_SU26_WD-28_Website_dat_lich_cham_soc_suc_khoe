# Notification Email UI Guide

The admin notification screen supports website and email sending modes.

## Email Mode

1. Enter the email subject.
2. Select recipient target.
3. Choose all users in a role or selected users when available.
4. Enter the email body.
5. Send email.

## Expected UI State

- The send button is disabled while submitting.
- Errors are shown without crashing the page.
- History rows render even when optional counters are missing.
